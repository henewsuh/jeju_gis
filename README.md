# 공간정보 탐색적 데이터 분석 경진대회
> ## Project Title  
>
> 제주도 재난지원금 사용건수 및 사용금액 비교 분석 (소상공인구분별/업종별/월별/시간대별)
>
> ## Motivation 
>
> 2회에 걸쳐 지급된 재난지원금의 사용금액과 이용건수를 다각도에서 세분화하여 분석하고자 함.
>
> 현재 v1 코드는 소상공인별/업종별/월별/시간대별 카테고리만 포함하고 있으며, 이후에 공간정보를 활용한 GIS 분석도 진행할 계획임. 
>
